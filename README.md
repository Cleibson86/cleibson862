Guia de consulta rápida de Markdown
Uma breve referência à sintaxe Markdown.

Visão geral
Este guia rápido de Markdown oferece uma visão geral de todos os elementos da sintaxe Markdown. Ele não abrange todos os casos extremos, portanto, se precisar de mais informações sobre algum desses elementos, consulte os guias de referência para sintaxe básica e sintaxe estendida .

Sintaxe básica
Esses são os elementos descritos no documento de design original de John Gruber. Todos os aplicativos Markdown são compatíveis com esses elementos.

Elemento	Sintaxe Markdown
Cabeçalho	# H1
## H2
### H3
Audacioso	**bold text**
itálico	*italicized text*
Citação em bloco	> blockquote
Lista ordenada	1. First item
2. Second item
3. Third item
Lista não ordenada	- First item
- Second item
- Third item
Código	`code`
Regra horizontal	---
Link	[title](https://www.example.com)
Imagem	![alt text](image.jpg)
Sintaxe estendida
Esses elementos ampliam a sintaxe básica, adicionando recursos extras. Nem todos os aplicativos Markdown são compatíveis com esses elementos.

Elemento	Sintaxe Markdown
Mesa	| Syntax | Description |
| ----------- | ----------- |
| Header | Title |
| Paragraph | Text |
Bloco de código cercado	```
{
  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}
```
Nota de rodapé	Here's a sentence with a footnote. [^1]

[^1]: This is the footnote.
ID do cabeçalho	### My Great Heading {#custom-id}
Lista de definições	term
: definition
Riscado	~~The world is flat.~~
Lista de Tarefas	- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media
Emoji
(veja também Copiar e Colar Emoji )	That is so funny! :joy:
Destaque	I need to highlight these ==very important words==.
Subscrito	H~2~O
Sobrescrito	X^2^
